# ArPixelForge - AI Text to Image Generator

ArPixelForge is an AI-powered text-to-image generation web app built with Next.js.  
It allows users to generate images based on natural language prompts using an AI image generation API.

---

## Live Demo
https://arpixelforge.vercel.app/

---

## Features
- Generate AI images from text prompts  
- Simple and fast interface built with Next.js and Tailwind CSS  
- Supports different image sizes or aspect ratios   
- Download or copy generated images  
- Responsive design for mobile and desktop  
- Uses an external AI API from Rapid API
